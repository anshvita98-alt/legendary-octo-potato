# **App Name**: Legacy Vault

## Core Features:

- WalletConnect Integration: Integrate WalletConnect for seamless connection with MetaMask, Coinbase Wallet, and other wallets on the Base blockchain, enabling secure transaction signing.
- NFT Minting: Enable users to upload legacy items (photos, stories, etc.), input metadata (title, description, family story), preview, and mint as secure ERC-721 NFTs on the Base chain with one-time pricing.
- Subscription Management: Implement a recurring subscription model (using onchain membership or integrated payments) that unlocks access to a personal dashboard with additional features.
- Gated Dashboard: Provide a subscription-only dashboard where users can view their minted heirlooms, edit metadata, manage private sharing links, and designate wallet heirs for inheritance.
- AI Assistant Chatbot: Embed an AI assistant chatbot as a helpful tool to guide users, generate stories from uploads, and answer questions related to the digital heirloom process.
- Admin Tools: Incorporate admin tools for managing one-time NFT minting flows vs subscription flows.
- Responsive Landing Page: An elegant landing page with explanation of digital heirlooms, a hero image, and call-to-action to connect wallet and mint NFTs.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) evoking royalty and timelessness.
- Background color: Dark grey (#212121) for a dark and elegant theme.
- Accent color: Lavender (#B39DDB) to highlight interactive elements and create visual interest.
- Body and headline font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel.
- Code font: 'Source Code Pro' for displaying any code snippets.
- Use elegant, minimalist icons to represent different types of legacy items and functionalities.
- Maintain a clean, organized layout with clear sections and intuitive navigation for easy exploration and management of digital heirlooms.
- Subtle animations and transitions to enhance user experience and evoke a sense of legacy and importance.
